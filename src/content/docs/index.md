title: "Bioinformatic Fridays"
---

Bioinformatic Fridays is a weekly, beginner-friendly training series for life scientists and bioinformatics trainees. Sessions combine short lectures with hands-on practice using real sequencing data and Linux command-line workflows.

## Course Structure
- Weekly, 2-hour live classes with guided demonstrations.
- Independent practice exercises between sessions.
- Materials are organized by module in this repository.

## How To Use This Repository
- Start in the module folder and follow lessons in order.
- Run the commands in your terminal and compare your output with the examples.
- Complete the exercises before checking the solutions.
